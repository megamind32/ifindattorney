## Color Scheme

**Primary Colors:**
- Red: `#dc2626` (red-600) - Buttons, borders on focus, selected states
- Red Gradient: `from-red-600 to-red-700` - Main CTA buttons
- Gray: `#111827` (gray-900) - Text
- Light Gray: `#f9fafb` (gray-50) - Background

**Interactive States:**
- **Hover:** Bright red border, light red background (`bg-red-50`)
- **Selected:** Red border, red background
- **Active:** Red border with shadow
- **Disabled:** Gray border, gray background, reduced opacity

**Typography:**
- Headings: Playfair Display (serif, italic, expressive)
- Body: Poppins (sans-serif, modern, clean)

---

## Animations

### Fade-In (Step Transitions)
- Duration: 0.4s
- Effect: Opacity 0 → 1, translateY 10px → 0
- Timing: ease-out

### Shake (Error Messages)
- Duration: 0.5s
- Effect: Oscillating horizontal movement
- Feedback: Clear visual error indication

### Hover Effects
- Buttons: Scale 105%, enhanced shadow
- Cards: Border color change, background tint, shadow lift
- Transitions: 300ms smooth duration

---

## Responsive Breakpoints

**Mobile (320px - 640px):**
- Single column layout
- Full-width inputs and buttons
- Smaller fonts (text-sm)
- Reduced padding

**Tablet (641px - 1024px):**
- Two-column grids where applicable
- Optimized spacing
- Medium fonts (text-base)

**Desktop (1025px+):**
- Multi-column layouts
- Max-width container (max-w-3xl)
- Larger heading fonts (text-4xl, text-5xl)
- Enhanced spacing and visual hierarchy

---

## Accessibility Features

✅ **Semantic HTML:**
- Proper form structure with `<form>` tag
- Labels associated with inputs via `htmlFor`
- ARIA attributes for error messages

✅ **Color Contrast:**
- Text on background meets WCAG AA standards
- Red accent color has sufficient contrast with white/light backgrounds

✅ **Keyboard Navigation:**
- Tab order follows logical flow
- Focus indicators visible on all interactive elements
- Enter key submits form

✅ **Error Handling:**
- Clear error messages
- Visual feedback (shake animation)
- Error prevention with required field validation

✅ **Mobile-Friendly:**
- Touch targets (min 44x44px)
- Readable text size (minimum 16px)
- Responsive design for all screen sizes
